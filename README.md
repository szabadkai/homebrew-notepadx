# homebrew-notepadx

Homebrew tap for [NotepadX](https://github.com/szabadkai/NotepadX) — a fast, beautiful, cross-platform text editor.

## Install

```bash
brew tap szabadkai/notepadx
```

### Cask (pre-built .app — recommended)

```bash
brew install --cask notepadx
```

### Formula (build from source)

```bash
brew install szabadkai/notepadx/notepadx
```

Requires the Rust toolchain. Build time is several minutes depending on your machine.

## Update

```bash
brew upgrade notepadx
```

## Uninstall

```bash
brew uninstall notepadx         # or brew uninstall --cask notepadx
brew untap szabadkai/notepadx
```
